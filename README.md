# StarWars Characters
Android App to to showcase Star Wars Character using https://swapi.dev/.

## Tech Stack.
- Kotlin
- MVVM
- Coroutines
- Paging 3 library
- Retrofit
- Flows
- Dagger Hilt for DI
- Navigation Components
